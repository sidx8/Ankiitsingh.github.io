Threading
Python threading allows you to have different parts of your program run concurrently and can simplify your design. A thread is a separate flow of execution. The different threads do not actually execute at the same time: they merely appear to. A daemon thread will shut down immediately when the program exits. One way to think is to consider the daemon thread a thread that runs in the background without worrying about shutting it down. If a program is running Threads that are not daemons, then the program will wait for those threads to complete before it terminates. Threads that are daemons, however, are just killed wherever they are when the program is exiting.
